bool diaballik::Board::isAntiJeu ( PlayerColor  color) const

Verify if there is a context of anti game. A context of anti game happends when the opponent player has a wall with his pieces and the current player has 3 pieces next to ennemy pieces.

Parameters
colorthe color of the current player.
Returns
true if there is a context of anti game so the current player can't go through the wall, otherwise false so when the wall do not exist or the current player has a piece behind the wall.
